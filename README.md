# vue-zoom
vue zoom component

# installing
`npm i @utlime/vue-zoom`

# simple example
```
<template>
  <vue-zoom>
    <img ... />
  </vue-zoom>
</template>
<script>
import { VueZoom } from '@utlime/vue-zoom';

export default {
  components: { VueZoom },
};
</script>

```
